นายโซลชาร์ รัตมูหมาด 6110110108 Section 02 

โปรแกรมนี้เป็นโปรแกรมสำหรับการฝึกภาษาอังกฤษ โดยแต่ละคำศัพท์จะเกิดจากการสุ่ม โดยในแต่ละคำศัพท์นั้นสามารถตอบได้ไม่เกิน 3 ครั้ง ถ้าเกิน 3 ครั้ง
โปรแกรมจะทำการเฉลยและรีเฟรชหน้าเว็ปเพจ ในกรณีที่ตอบผิดจะมี Alert เด้งขึ้นมาแจ้งเตือนแก่ผู้ใช้ และถ้าหากเป็นกรณีที่ตอบถูกจะมี Alert เด้งขึ้นมาแจ้งเตือนแก่ผู้ใช้
และทำการรีเฟรชหน้าเว็ปเพจไปยังคำศัพท์ข้ออื่นต่อไป

โดยสิ่งที่ทำสามารถสรุปตามการ Commit ดังนี้
1. Update App.js : สร้าง UI ให้มีความสวยงามแบ่ง Layout เพิ่มข้อความต่างๆ และจัดการด้วย App.css สำหรับการตกแต่ง
2. Add reload button : สร้างปุ่มสำหรับรีโหลดหน้าเว็ปเพจในกรณีที่ไม่สามารถตอบคำศัพท์ได้ ผู้ใช้สามารถกดปุ่มดังกล่าวเพื่อรีโหลดไปยังศัพท์ข้ออื่น
3. Add win lose alert : สร้าง Alert สำหรับการแจ้งเตือนสถานะ ในกรณีที่ตอบผิด และถ้าหากเป็นกรณีที่ตอบถูกจะมี
4. Add word : เพิ่มคำศัพท์ต่างๆ ให้มากยิ่งขึ้น โดยในการทำการทดลองครั้งนี้ จะมีศัพท์ทั้งหมด 10 คำศัพท์
5. Show answer and reload : จะทำการแสดงคำศัพท์ เมื่อผู้ใช้ไม่สามารถตอบได้เกิน 3 ครั้ง และจะทำการรีโหลดหน้าเว็ปเพจ
